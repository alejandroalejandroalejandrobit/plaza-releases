# Plaza — Releases

Official signed release builds of **Plaza**, the sovereign, non-custodial
Bitcoin Lightning point-of-sale app. Zero fees, forever.

This repository holds **release binaries only** — the source code lives in a
private repository.

## Install on Android

**Recommended — auto-updates via [Obtainium](https://github.com/ImranR98/Obtainium):**

1. Install Obtainium (from its GitHub releases or F-Droid/IzzyOnDroid).
2. Add App → paste this repo's URL:
   `https://github.com/alejandroalejandroalejandrobit/plaza-releases`
3. Obtainium tracks new releases and updates Plaza automatically.

**Manual:**

1. Open [Releases](../../releases/latest) and download the latest `plaza-*.apk`.
2. Tap the file and allow installation from your browser/files app when prompted.
